# Healthcare App

## About the Project

Healthcare App is an Android application developed to improve communication between patients and doctors. The application provides a platform where patients can book appointments online and offline, consult doctors, and understand medical reports with the help of AI, this application also provides additional features like nearbyHospital search.

The goal of this project is to make healthcare services more accessible and organized through a single mobile application.

---

## Features

### Patient Module
- Register and Login using Email and Password
- Book Online and Offline Appointments
- View Appointment Details
- AI-Based Medical Report Explanation
- Search Nearby Hospitals
